**MCPX** is a marketplace, gateway, and x402 facilitator for
[Model Context Protocol](https://modelcontextprotocol.io) servers, built
Sui-native. Every tool call settles on-chain in **USDsui** in a single
atomic transaction; the receipt is a **soulbound on-chain object**; the
developer is paid straight to a **Sui DeveloperVault** the same block.

No credit cards. No platform tokens. Stablecoin revenue from day one.

## Why it exists

The MCP ecosystem has thousands of servers and no native way to pay for
them. MCPX makes a paid tool call a first-class on-chain primitive:

- **Atomic settlement.** Debit the user session → credit the dev vault →
  credit the treasury → credit the insurance pool → mint the receipt. All
  in one programmable transaction block. If anything fails, nothing moves.
- **Permanent receipts.** Each `CallReceipt` is `key`-only (no `store`) —
  it can never be transferred after mint. The output log lives on Walrus,
  permanently retained.
- **On-chain economics.** The 2.5% take rate (0.5% insurance + 2.0%
  treasury) lives in a shared `PlatformConfig` object, mutable only by an
  admin multisig, hard-capped at 10%.
- **Multi-chain ready, Sui-only today.** Every chain-touching call goes
  through one `ChainAdapter`. Adding Base or Solana later is additive.
